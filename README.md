# TRACK
TRACK: transient checking and filtering pipeline for Long period transients in GMRT archival data

## Summary
Find transients using MAD based thresholding for all the image cubes in given directroy using central pixel lightcurve. 
For the cube in which traneints are dteceted, a lightcurve PNG will be saved
and for every transient a GIF will be saved
